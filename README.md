### Philosophy

- **Test everything.** If it's not tested, it's not production-ready.
- **Document the why, not just the what.** Architecture decisions matter more than code comments.
- **Immutable raw layers.** Never mutate source data. Always reprocess from scratch.
- **Fail loudly.** Silent failures are worse than loud crashes.
